wtgl开发blog

发表于 2023-09-03

这是一篇介绍梧桐故里开发的blog，这算是我第一次多人合作进行项目，积累了一定的多人合作经验，目前而言经验都是来于教训，总体而言开发过程比较痛苦，但这也让我对多人开发项目制定相关规范性产生了更深的体会。目前还没有开发完成，但剩下的都是重复性的工作了，所以可以先把开发的blog写下来。

梧桐故里项目介绍

梧桐故里项目是基于WebGal引擎开发的文字交互游戏，具体的技术并不难，主要是面向惠州中学毕业高中生所开发的游戏，以具有代表性的惠中生活为剧情，勾起惠中高中生对于母校的思念之情。

梧桐故里项目部署渠道

因为该引擎有文件导出功能，支持三种渠道的部署
1.Windows端部署:inno setup compiler实现
2.Android端部署：Android Studio实现
3.网页端部署：运用宝塔平台实现

开发过程的收获

1.每使用一个全新软件，都需要去访问它的官方文档，里面有着全面的诠释和使用方法
2.进行多人开发时，要根据自我开发的需求提交自己的开发意见，以减少工作量，本次多人合作开发，资料非常的乱，文本不根据框架走，文本内部结构不统一规范等等，使得开发难度大大提升，造成了许许多多的困难和挑战。
3.csdn的认识，在未进行开发之前，我对于csdn的用法其实还不太熟悉，但当面对着许许多多的困难时，我终于意识到csdn平台的重要性，在这之前，我只会跟着B站的教程走，或许这也是进步吧。

blog搭建

欢迎来到我的博客空间，这应该是我严格意义的第一篇博客，尽管之前也试过用WordPress直接在服务器上搭建博客，但我个人不太喜欢
那种搭建方式，所以并没有生产内容，所以这应该是我第一篇有内容的博客了。
本篇博客主要围绕着搭建博客空间所用到的一些知识和程序进行梳理。目前我对于博客的Markdown的使用还不太熟悉，需要一段学习时间，所以你可以看到我的格式会比较稚嫩。

所部署的软件与技术

1.git:版本管理工具
2.node.js(目前作用未知)
3.hexo:静态博客生成工具

对于git我的使用理解并不深，目前来看就是对可以用程序化的语言对文件进行管理，但它所谓的版本控制性目前我还未进行掌握。

hexo

node.js

目前没有更进一步的想法

SSH

应该涉及到计算机网络的知识，可以去简单的弄清原理

github

github的使用目前我还是不太熟悉，我希望多接触几个项目，慢慢认识这个平台。
